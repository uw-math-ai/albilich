Proposed obstruction lemma. Let n >= 7, and let x and y have orders 2 and 3 in S_n, respectively. Write the cycle types as 2^m 1^f and 3^k 1^e. If f,e > 0, both types have subset sum 1. If f > 0 and e=0, the x-type has subset sum 3: use three fixed points when f >= 3, and one transposition plus one fixed point when f is 1 or 2; the y-type has a 3-cycle. If f=0 and e >= 2, both have subset sum 2. If f=0 and e=1, then n is even and congruent to 1 modulo 3; for n >= 7 this forces n >= 10, and both types have the proper subset sum 4. If f=e=0, then 6 divides n; since n >= 7, the common subset sum 6 is proper. Thus in every case there is 0 < d < n representable as a union of cycles of each element. Suitable conjugates of x and y preserve the same d-set and cannot generate S_n. For A_n, an involution has even m, and for n >= 7 none of the relevant order-2 or order-3 S_n conjugacy classes splits in A_n; the same placement argument therefore gives conjugates inside A_n preserving a common proper subset. Hence neither S_n nor A_n is invariably generated by an involution and an element of order 3 for n >= 7. This blocks the canonical alternating/symmetric carrier route but does not refute the root, since other finite overgroups may exist.
