# Exact integral reduction and a necessary correction

# Direct root attack and remaining obstruction

The criterion above therefore closes the universal fixed-field module argument but does not itself decide existence of a gap-four pair. # Self-check

The proof uses finite F so that R/J_L is a finite elementary abelian group; it uses right modules consistently; it does not assume V is irreducible, faithful, or semisimple; quotient derived lengths supply both lower bounds; maximality is obtained by full inverse image. The integral counterexample attacks only the tensor shortcut, not the fixed-characteristic criterion. A decisive CAS pass would use GAP or Sage on one explicitly presented seed, compute integer generator matrices for P_Z and J_Z, determine generic rational ranks and exceptional primes from nonzero minors, and compare their modular ranks at every exceptional prime.
