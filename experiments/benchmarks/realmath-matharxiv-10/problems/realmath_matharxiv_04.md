# RealMath problem realmath_matharxiv_04

Source: http://arxiv.org/abs/2412.02902v1

Determine the number of rising‐continuous functions \(\chi:\mathbb{Z}_p\to\mathbb{C}_{q_H}\) that satisfy the functional equation
\[
\chi(p\mathfrak{z}+j)=\frac{\mu_j}{p}\chi(\mathfrak{z})+c_j, \quad \forall j\in\{0,1,\ldots,p-1\},\,\forall \mathfrak{z}\in\mathbb{Z}_p.
\]
for a semi-basic \(p\)-Hydra map \(H\) fixing \(0\) (with \(\mu_j/p=H'_j(0)\)).
